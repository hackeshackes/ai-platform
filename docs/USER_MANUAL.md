# AI Platform 用户手册

## 快速开始

访问: http://localhost:3000
账号: admin / admin123

## 功能说明

### 项目管理

- 创建/编辑/删除项目
- 搜索过滤

### 任务管理

- 查看任务列表
- 筛选状态
- 查看详情和日志

### GPU监控

- 实时显存
- 利用率图表
- 温度监控

### 训练任务

4步向导:
1. 选择模型
2. 选择数据集
3. 配置训练
4. 提交

### 推理服务

1. 选择模型
2. 输入提示词
3. 设置参数
4. 生成结果
